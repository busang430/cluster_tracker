# 42 Cluster Time Tracker - 安装指南

## 📋 安装前准备

在安装扩展之前，你需要准备图标文件。

### 创建图标（重要！）

扩展需要三个图标文件，请按照以下步骤创建：

1. 打开 `icons/README_ICONS.md` 查看详细的图标创建说明
2. 创建或下载三个PNG图标：
   - `icon16.png` (16x16像素)
   - `icon48.png` (48x48像素)
   - `icon128.png` (128x128像素)
3. 将图标文件放在 `extension/icons/` 文件夹中

**临时方案**：如果暂时没有图标，可以使用任意PNG图片重命名后使用，不影响扩展功能。

## 🚀 安装步骤

### 1. 打开Chrome扩展管理页面

在Chrome浏览器地址栏输入：
```
chrome://extensions/
```

### 2. 开启开发者模式

在页面右上角找到"开发者模式"开关，将其打开。

### 3. 加载扩展

1. 点击左上角的"加载已解压的扩展程序"按钮
2. 在文件选择器中，导航到：
   ```
   d:\42\cluster\extension
   ```
3. 选择该文件夹并点击"选择文件夹"

### 4. 确认安装

如果一切正常，你会看到：
- ✅ 扩展卡片出现在列表中
- ✅ 扩展名称：42 Cluster Time Tracker
- ✅ 版本：1.0.0
- ✅ 状态：已启用

## ✅ 验证安装

### 检查扩展是否正常工作

1. **查看工具栏图标**
   - Chrome工具栏应该出现扩展图标
   - 如果没有，点击拼图图标📌，将扩展固定到工具栏

2. **访问Matrix页面**
   - 打开 https://matrix.42lyon.fr/
   - 登录你的42账号
   - 页面右上角应该出现时间追踪面板

3. **测试弹出窗口**
   - 点击工具栏的扩展图标
   - 应该弹出统计窗口

## ⚠️ 常见问题

### 问题1：扩展无法加载

**错误提示**："清单文件缺失或不可读"

**解决方法**：
- 确保选择的是 `extension` 文件夹，不是 `cluster` 文件夹
- 检查 `manifest.json` 文件是否存在

### 问题2：图标显示错误

**错误提示**："无法加载图标"

**解决方法**：
- 检查 `icons` 文件夹是否存在
- 确保三个图标文件都已创建
- 临时方案：使用任意PNG图片重命名

### 问题3：页面没有显示追踪面板

**可能原因**：
- 没有登录42账号
- 不在Matrix页面
- 扩展权限未授予

**解决方法**：
1. 确保已登录 https://matrix.42lyon.fr/
2. 刷新页面（F5）
3. 打开开发者工具（F12）查看Console是否有错误

### 问题4：无法追踪时间

**可能原因**：
- Cookie未授权
- 网络连接问题

**解决方法**：
1. 在 `chrome://extensions/` 页面找到扩展
2. 点击"详细信息"
3. 确保"在所有网站上"权限已启用
4. 刷新Matrix页面

## 🔧 开发者调试

如果需要调试扩展：

1. **查看后台日志**
   - 在扩展卡片上点击"service worker"
   - 打开后台控制台

2. **查看Content Script日志**
   - 在Matrix页面按F12
   - 查看Console标签

3. **重新加载扩展**
   - 修改代码后，点击扩展卡片上的刷新图标🔄
   - 或者关闭再重新加载扩展
