# 图标说明

由于图标生成服务暂时不可用，你需要手动创建以下三个图标文件：

## 需要创建的图标

1. **icon16.png** - 16x16像素
2. **icon48.png** - 48x48像素  
3. **icon128.png** - 128x128像素

## 设计建议

- **背景**：使用紫色渐变（从 #667eea 到 #764ba2）
- **图标**：白色的秒表⏱️或时钟图标
- **可选**：在图标上加上"42"数字
- **风格**：简洁、现代、扁平化设计

## 快速创建方法

### 方法一：使用在线工具
1. 访问 https://www.canva.com/ 或 https://www.figma.com/
2. 创建对应尺寸的画布
3. 添加紫色渐变背景
4. 添加白色秒表emoji或图标
5. 导出为PNG格式

### 方法二：使用Photoshop/GIMP
1. 创建对应尺寸的新文件
2. 填充紫色渐变背景
3. 添加白色秒表图标
4. 保存为PNG格式

### 方法三：使用简单的占位图标
暂时可以使用任意PNG图片作为占位符，扩展功能不受影响。

## 文件位置

将创建好的图标文件放在：
- `d:\42\cluster\extension\icons\icon16.png`
- `d:\42\cluster\extension\icons\icon48.png`
- `d:\42\cluster\extension\icons\icon128.png`

## 临时解决方案

如果暂时没有图标，可以：
1. 从网上下载任意PNG图标
2. 重命名为对应的文件名
3. 放在icons文件夹中
4. 扩展仍然可以正常工作

图标只影响视觉效果，不影响功能！
